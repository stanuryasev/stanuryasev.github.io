CASE STUDY SUMMARYn This case study solves several Travelling Salesman Problems (TSPs). PSG has TSP operator for a compact formulation of problems. The length of the path which goes through all vertices of the graph is calculated with the operator “linear(TSP(matrix_of_distances))”. TSP optimization problems can be solved only if Gurobi package is installed at the computer. PSG is using Gurobi-based solvers CarGrb or VanGrb for solving problems with TSP operator. Problems were solved with data downloaded from the link [1].
References [1] https://comopt.ifi.uni-heidelberg.de/software/TSPLIB95/.
